## Starten (XAMPP)

- Zet de map in `xampp/htdocs/` (staat al goed).
- Start Apache in XAMPP.
- Open in je browser: `http://localhost/cursorkiosk/`

## Design assets

- Afbeeldingen staan in `assets/img/`.
- Kleuren en typografie staan in `css/styles.css`.

## Renos-Rough font

Plaats de font als bestand:

- `assets/fonts/Renos-Rough.ttf`

Daarna gebruikt de UI automatisch Renos-Rough voor headings (met fallback naar Noto Sans).
